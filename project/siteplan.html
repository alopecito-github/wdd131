Baker Montreal Condo Rentals Hub - Site Plan

Site Purpose

The site will promote rental listings in Montreal, all listed by my company's brokers. The name reflects the location, the company, and the purpose of the website. Users will be able to fill out rental applications, contact us to express interest based on JS powered filters, also will allow listing updates based on availability with inquiry forms

User Scenarios

Scenarios for Montreal-area users looking for a long-term Condo to rent

  • How can I filter condos by price or neighborhood?
  • Which condos are currently available, prices and sizes?
  • How do I apply for a unit?

Color Scheme

The color palette for this project is inspired by the branding used by Baker Real Estate. Their visual style focuses on a luxury and high-contrast colors using deep navy, gold accents, and a strong red for call-to-action buttons.

Color Hex Value Description
#0b1c2c Primary (Navy): Used for headings, main titles
#da291c Secondary (Red): Used for headings and important actions such as “Apply Now” buttons, hover effects and alerts.
#c59d5f Accent (Gold): Used for highlights, lines, decorative accents, and subtle emphasis.
#f8f5f2 Light Background: Used for page backgrounds and section areas to improve readability.
#333333 Text Color: Used for regular body text for optimal contrast and readability.

Typography

The typography of my website is inspired by the branding used by Baker Real Estate. Their primary fonts are Denton (for headings) and Neue Haas Grotesk Display Pro (for body text). Since these are licensed typefaces, this project uses high-quality safe google fonts equivalents:

  • Headings: "Instrument Serif", serif (inspired by Denton)
  • Body Text: "Inter", sans-serif (inspired by Neue Haas Grotesk Display Pro)
